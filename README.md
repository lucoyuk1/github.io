# 💕 我们的100天纪念册

一个浪漫的纪念日网页，用于庆祝你们的100天。采用粉色系设计，支持密码保护，包含多页故事展示和3D相册效果。

## ✨ 功能特点

- **密码保护**：访问页面需要输入密码（5201010）
- **9页精美设计**：
  1. 封面页 - 浪漫开场
  2. 倒计时页 - 显示相识至今的天数
  3-7. 照片故事页 - 展示你们的回忆
  8. 3D旋转相册 - 炫酷的3D效果
  9. 告白页 - 表达爱意
- **完美适配手机**：响应式设计，手机端体验更佳
- **漂浮爱心动画**：浪漫的装饰效果
- **触摸滑动**：支持左右滑动切换页面
- **页面指示器**：清晰显示当前页面位置

## 📸 如何添加照片

### 方法1：将照片放到 images 文件夹（推荐）

1. 将你的照片重命名为以下名称：
   - `photo1.jpg` - 初次相遇的照片
   - `photo2.jpg` - 第一次约会的照片
   - `photo3.jpg` - 美好时光的照片
   - `photo4.jpg` - 难忘回忆的照片
   - `photo5.jpg` - 更多回忆的照片

2. 3D相册需要12张照片：
   - `album1.jpg` 到 `album12.jpg`

3. 将这些照片复制到 `images` 文件夹中

### 方法2：使用图床链接

如果你使用在线图床（如 pic.img.wang），可以直接修改 HTML 中的图片链接。

## ⚙️ 自定义设置

### 修改密码

在 `index.html` 第767行，修改密码：

```javascript
const PASSWORD = "5201010"; // 改成你想要的密码
```

### 修改相识日期

在 `index.html` 第970行，修改开始日期：

```javascript
const startDate = new Date('2024-08-25'); // 改成你们的相识日期
```

### 修改页面文字

你可以直接编辑 HTML 中的文字内容：

- **封面标题**（第629行）：`<h1 class="cover-title">我们的100天</h1>`
- **各页标题和内容**：在第648-755行之间
- **告白内容**（第737-753行）：修改成你想说的话

### 修改配色方案

如果想要其他颜色主题，可以修改以下颜色值：

- 粉色系：`#ffd1dc`, `#ffb6c1`, `#ff9eb5`, `#ff69b4`
- 可以搜索这些颜色代码并替换成你喜欢的颜色

## 🚀 部署到 GitHub Pages

1. 确保所有照片都已添加到 `images` 文件夹
2. 提交所有文件到 GitHub 仓库
3. 在 GitHub 仓库设置中启用 Pages
4. 访问 `https://你的用户名.github.io/` 查看效果

## 📱 使用说明

### 电脑端

- 点击左右箭头按钮切换页面
- 或使用键盘 ← → 方向键
- 顶部有页面指示器显示当前位置

### 手机端

- 左右滑动切换页面
- 点击底部箭头按钮也可切换
- 3D相册页面点击可展开照片

## 💡 提示

1. **照片格式**：支持 jpg、png、gif 等常见格式
2. **照片大小**：建议宽度在 600-1200px 之间，单张不超过 2MB
3. **照片比例**：横向照片效果最佳（16:9 或 4:3）
4. **如果照片未上传**：页面会显示占位图，不会影响使用
5. **清除密码记录**：如果想重新输入密码，清除浏览器 localStorage 即可

## 🔧 技术栈

- 无需任何框架或依赖
- 响应式设计（Mobile First）
- CSS3 动画和 3D 变换
- LocalStorage 密码缓存
